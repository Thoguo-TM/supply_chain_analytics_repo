# Phase 1 — Excel Essentials (Weeks 1–8)
**Goal:** Master Excel for supply chain data cleaning, basic analytics, and reporting.

## Daily Grind

### Week 1 — Excel Fundamentals
- **Day 1:** Open Week 1 Sales dataset, inspect structure, note issues.
- **Day 2:** Learn Excel data types (number, text, date), format dataset columns correctly.
- **Day 3:** Practice cell referencing (absolute vs relative), apply in small formulas.
- **Day 4:** Use basic formulas: SUM, AVERAGE, MIN, MAX on sales data.
- **Day 5:** Apply sorting & basic filtering to isolate Widget A sales.
- **Day 6:** Practice conditional formatting: highlight sales > 20 units.
- **Day 7:** Weekly recap — Document in notebook what was learned & cleaned dataset.

### Week 2 — Core Formulas
- **Day 8:** Practice IF statements for flagging low stock.
- **Day 9:** Use VLOOKUP to link sales to inventory.
- **Day 10:** Learn INDEX + MATCH for more flexible lookups.
- **Day 11:** Use COUNTIF and SUMIF for product sales totals.
- **Day 12:** Apply text functions: LEFT, RIGHT, MID, TRIM, PROPER.
- **Day 13:** Create a product performance summary table.
- **Day 14:** Weekly recap — Save cleaned & summarized dataset.

### Week 3 — Data Cleaning
- **Day 15:** Identify & remove duplicates from raw data.
- **Day 16:** Handle missing values (replace or flag).
- **Day 17:** Split concatenated fields using Text-to-Columns.
- **Day 18:** Use Find & Replace for quick fixes.
- **Day 19:** Apply data validation to prevent wrong inputs.
- **Day 20:** Standardize product names & SKUs.
- **Day 21:** Weekly recap — Document cleaning steps in README.

### Week 4 — Pivot Tables
- **Day 22:** Build a simple pivot table for sales by product.
- **Day 23:** Add filters and slicers to pivot table.
- **Day 24:** Group dates into months & quarters.
- **Day 25:** Calculate % of total sales in pivot table.
- **Day 26:** Add calculated fields in pivot tables.
- **Day 27:** Create pivot chart linked to pivot table.
- **Day 28:** Weekly recap — Export pivot table report.

### Week 5 — Dashboards
- **Day 29:** Learn basic dashboard layout principles.
- **Day 30:** Insert charts: bar, line, and pie for sales trends.
- **Day 31:** Link charts to dynamic named ranges.
- **Day 32:** Add KPI cards with formulas (e.g., Total Sales).
- **Day 33:** Use form controls (dropdowns) for filtering.
- **Day 34:** Format dashboard professionally.
- **Day 35:** Weekly recap — Save dashboard file.

### Week 6 — Advanced Excel Functions
- **Day 36:** Use XLOOKUP as a modern alternative to VLOOKUP.
- **Day 37:** Apply nested IF statements for multi-tier logic.
- **Day 38:** Learn and use the FILTER function.
- **Day 39:** Apply dynamic array functions (UNIQUE, SORT).
- **Day 40:** Use AGGREGATE for advanced summarization.
- **Day 41:** Create running totals and moving averages.
- **Day 42:** Weekly recap — Publish advanced formulas sheet.

### Week 7 — Case Study Project
- **Day 43:** Import a messy supplier dataset.
- **Day 44:** Clean dataset for duplicates & missing data.
- **Day 45:** Create pivot summary for supplier delivery times.
- **Day 46:** Build KPI dashboard for supplier performance.
- **Day 47:** Present findings in a summary sheet.
- **Day 48:** Review all Week 1–6 techniques.
- **Day 49:** Weekly recap — Submit case study.

### Week 8 — Final Excel Project
- **Day 50–56:** Build a **Mini Supply Chain Dashboard**:
  - Integrate sales, inventory, and supplier datasets.
  - Create automated stock level alerts.
  - Add product profitability chart.
  - Build interactive controls for filtering.
  - Deliver as Excel file + PDF report.
