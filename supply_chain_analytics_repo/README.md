# Supply Chain Analytics Repo

## Structure
This repo is organized into phases, each with its own folder for exercises and projects:
- phase1_excel ? Weeks 1–8
- phase2_sql ? Weeks 9–14
- phase3_python ? Weeks 15–27
- phase4_powerbi ? Weeks 28–36
- phase5_advanced_analytics ? Weeks 37–45
- phase6_special_topics ? Weeks 46–52
- phase7_capstone ? Weeks 53–70

## Getting Started
1. Open this repo in VS Code.
2. Install dependencies with:
   pip install -r requirements.txt
3. Start with phase1_excel for Week 1 exercises.
4. Original datasets live in data/raw/.
5. Save cleaned datasets in data/cleaned/.
